Developer Guide

Dokumentasi Integrasi

Selamat datang di pusat referensi Nusantara API. Panduan ini dirancang untuk membantu Anda menghubungkan aplikasi Anda dengan database kuliner kami dalam hitungan menit.

1. Autentikasi (API Key)

Keamanan adalah prioritas kami. Setiap permintaan (Request) ke API wajib menyertakan API Key yang valid. Key ini dikirim melalui Header HTTP.

Anda dapat melihat atau membuat API Key baru di halaman Dashboard Developer (Perlu Login).

// Format Header Request
x-api-key: sk-or-v1-xxxxxxxxxxxxxxxxxxxx

2. Mengambil Data Resep

Gunakan endpoint di bawah ini untuk mendapatkan daftar seluruh resep masakan yang tersedia di database kami.

GET /api/v1/recipes

Contoh Response JSON:

[ { "id": 1, "judul": "Rendang Sapi", "deskripsi": "Masakan daging bercita rasa pedas...", "gambar": "http://localhost:3000/uploads/rendang.jpg", "bahan": "Daging sapi, Santan, ...", "langkah": "1. Rebus daging..." }, ... ]

3. Contoh Implementasi (JavaScript)

Berikut adalah contoh kode sederhana menggunakan fetch untuk menampilkan data resep di console browser Anda.

const apiKey = 'MASUKKAN_KEY_ANDA_DISINI'; async function getResep() { try { const response = await fetch('http://localhost:3000/api/v1/recipes', { headers: { 'x-api-key': apiKey } }); const data = await response.json(); console.log("Data Diterima:", data); } catch (error) { console.error("Error:", error); } } getResep();
